这是用hexo+github搭建博客的第一篇文章：）

遇到的一些坑：

1._config.yml里

url: http://www.zhangkun.site/  #这里填你申请的域名
root: /                            #这里填根目录就好

2.CNAME文件

zhangkun.site                    #直接写申请的域名就好，不要带协议和次级域名

（直接 在/source/CNAME配置 deploy后自动在github根目录生成~）

更换主题步骤:

1.找到你要换的主题的github地址，终端里输入

$ git clone https://github.com/iissnan/hexo-theme-next.git themes/next

2.在_config.yml里修改

theme: next
